# 투자 포트폴리오 대시보드

Streamlit을 이용한 주식 포트폴리오 분석 대시보드입니다.

## 기능

### 첫 번째 탭: 포트폴리오 분석
- 팀/자산/섹터별 기업 분류
- 주가 데이터 및 재무 지표 분석
- 실시간 수익률 계산
- 개별 종목 상세 차트

### 두 번째 탭: 트렌드 분석
- 팀별 평균 변동률 비교
- 섹터별 평균 변동률 비교
- 섹터별 개별 종목 변동률 분석

## 설치 방법

### 로컬 실행

```bash
# 패키지 설치
pip install -r requirements.txt

# 앱 실행
streamlit run app.py
```

### Streamlit Cloud 배포

1. GitHub에 이 저장소를 업로드
2. [Streamlit Cloud](https://streamlit.io/cloud)에 접속
3. "New app" 클릭
4. 저장소 선택 및 배포

## 사용 방법

1. **기간 설정**: 좌측 사이드바에서 분석 시작일과 종료일을 선택
2. **Y축 범위 조정**: 차트의 Y축 범위를 필요에 따라 조정
3. **분석 시작**: "분석 시작" 버튼 클릭
4. **결과 확인**: 
   - 첫 번째 탭에서 전체 포트폴리오 현황 확인
   - 개별 종목 선택하여 상세 차트 확인
   - 두 번째 탭에서 팀/섹터별 트렌드 분석

## 주요 지표

- **기준가**: 시작일의 종가
- **최고가**: 기간 중 최고가
- **현재가**: 종료일의 종가
- **누적수익률**: 기준가 및 최고가 대비 수익률
- **일일수익**: 전일 대비 가격 변동
- **부채비율**: 부채/자기자본 비율
- **유동비율**: 유동자산/유동부채 비율
- **ROE**: 자기자본이익률
- **Runway**: 현금 소진까지 예상 기간(년)
- **Total Cash**: 현금 및 단기투자자산
- **FCF**: 영업현금흐름 - 자본지출

## 주의사항

- 데이터 로딩에 시간이 걸릴 수 있습니다 (약 200+ 종목)
- API 호출 제한으로 인해 일부 데이터가 표시되지 않을 수 있습니다
- 실시간 데이터가 아닌 지연된 데이터입니다
